# 📝 Python To-Do List

## 📌 Project Overview
This is a simple To-Do List application developed in Python as part of the DecodeLabs Python Internship Project 1.

The application allows users to manage their daily tasks using a menu-driven program.

## 🚀 Features
- Add a new task
- View all tasks
- User-friendly menu
- Exit the application safely

## 🛠️ Technologies Used
- Python 3
- Lists
- Loops
- Conditional Statements
- Functions (if used)

## ▶️ How to Run
1. Install Python 3.
2. Download the project.
3. Open the project folder.
4. Run the following command:

```bash
python todo_list.py
```
